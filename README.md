# 🔢 Number Analyzer (Python)

A Python program that performs multiple operations on a given number, such as checking even/odd, reversing digits, and more.

# 💡 Overview

This project was built as part of my Python learning journey.
It combines multiple number-based operations into a single program to practice logic building and problem-solving.

# ⚙️ Features
* 🔍 Even / Odd check
* ➕ Sum of digits
* 🔢 Count total digits
* 🔄 Reverse the number
* 🔁 Palindrome check
* 📈 Find largest digit

# 🛠️ Technologies Used
* Python

# 📚 What I Learned
* Working with loops
* Using conditional statements
* Breaking problems into smaller parts
* Handling numbers and digits

# 🎯 Purpose
This is a beginner-level project created to strengthen core Python concepts and improve logical thinking.
